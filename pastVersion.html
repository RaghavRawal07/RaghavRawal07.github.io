Sticky Elements - Advanced Positioning

CSS sticky positioning creates elements that stick to a specific position during scrolling. They behave like relatively positioned elements until they reach a threshold, then become fixed. This is perfect for navigation bars, sidebars, and table headers.

Lightning Web Components - Salesforce Modern Development

Lightning Web Components (LWC) is Salesforce's modern programming model for building lightning components. It leverages web standards like Custom Elements, Templates, Decorators, ES6+ modules, and Shadow DOM.

LWC Architecture and Examples

Basic LWC Component Structure

// helloWorld.js import { LightningElement, track } from 'lwc'; export default class HelloWorld extends LightningElement { @track greeting = 'Hello'; @track name = 'World'; handleNameChange(event) { this.name = event.target.value; } get displayMessage() { return `${this.greeting}, ${this.name}!`; } }
<!-- helloWorld.html --> <template> <div class="container"> <h1>{displayMessage}</h1> <lightning-input label="Enter your name" value={name} onchange={handleNameChange}> </lightning-input> </div> </template>

LWC Lifecycle Hooks

import { LightningElement } from 'lwc'; export default class LifecycleExample extends LightningElement { connectedCallback() { // Component inserted into DOM console.log('Component connected'); } renderedCallback() { // After component renders console.log('Component rendered'); } disconnectedCallback() { // Component removed from DOM console.log('Component disconnected'); } errorCallback(error, stack) { // Error boundary console.error('Component error:', error); } }

LWC Data Binding & Events

// Parent Component import { LightningElement, track } from 'lwc'; export default class ParentComponent extends LightningElement { @track items = ['Apple', 'Banana', 'Orange']; handleItemSelect(event) { const selectedItem = event.detail.item; console.log('Selected:', selectedItem); } }
// Child Component import { LightningElement, api } from 'lwc'; export default class ChildComponent extends LightningElement { @api items = []; handleClick(event) { const item = event.target.dataset.item; const selectEvent = new CustomEvent('itemselect', { detail: { item } }); this.dispatchEvent(selectEvent); } }

Key LWC Concepts:

  • @track: Makes properties reactive
  • @api: Exposes public properties
  • @wire: Connects to Salesforce data
  • Custom Events: Component communication
  • Lightning Data Service: Cached data access
  • Apex Integration: Server-side logic

Benefits: Modern web standards, better performance, smaller bundle size, improved developer experience, and seamless Salesforce integration.
