Shadow DOM Demo

Two sections, two custom elements, two shadow roots.

Open DevTools and expand each custom element to see #shadow-root (open).

Section 1

Always-visible Shadow DOM

The content below is rendered inside a shadow root as soon as the page loads. The global CSS on this page tries to paint every <p> bright red, but the scoped styles inside the shadow root win.

Section 2

Click-to-reveal Shadow DOM

The button below lives inside a shadow root. Clicking it injects more nodes into the same shadow root, so the revealed content is also shadow DOM.

Section 3

Plain iframe (no Shadow DOM)

This <iframe> is rendered directly in the light DOM. It is not wrapped in any custom element and has no shadow root — inspect it in DevTools and you will see the <iframe> tag sitting right inside <main>.

Built with vanilla HTML, CSS, and Web Components — no frameworks.
